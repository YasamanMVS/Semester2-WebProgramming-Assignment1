# COMP1231 Assignment 1
## Description:
This project is a simple interactive web application created for the COMP1231 course. It utilizes HTML, CSS, and JavaScript to collect and display user input. The application prompts the user for their name, program of study, and year of study. After providing valid inputs, the user sees their details displayed on the webpage.

## Installation:
No installation is necessary. To view and interact with the website, follow these steps:

1. Download the project files to a local directory.
2. Open the `welcome.html` file in any modern web browser to start the application.

## Features:
- **User Interaction**: Uses JavaScript prompts to collect user data.
- **Validation**: Ensures that the user inputs are not empty and meet certain criteria (e.g., year of study must be a number between 1 and 3).
- **Responsive Design**: Styled with CSS for a clean and simple user interface.

## Usage:
Upon opening the welcome.html file in a browser:

1. The user is greeted with a JavaScript alert introducing the application.
2. The user is prompted to enter their name, program, and year of study one by one.
3. If the input is valid, it is displayed under the respective headers on the webpage.

## Technologies Used:
- **HTML5**: For structuring the content.
- **CSS3**: For styling the webpage.
- **JavaScript**: For dynamic interactions and validating user input.
